# 基于 React 的论坛系统

本项目是基于 Express + SQLite + React + React Hooks + Redux + React Router + Ant Design + Axios + Sass + ES6 等技术栈开发的一个论坛系统，UI 界面参考了 V2EX

## 项目功能实现

- [x] 首页全部帖子及发布时间展示
- [x] 帖子分类及评论数量展示
- [x] 用户注册（头像上传到静态服务器）
- [x] 用户登录
- [x] 发布新帖子
- [x] 回复帖子
